O objetivo de fazer uma calculadora de horas em React é basicamente aprender a criar uma aplicação interativa. A gente vai usar o React para construir a interface, controlar o que aparece na tela e como as horas são calculadas. A ideia é entender como gerenciar o estado dos componentes, fazer a aplicação responder às ações do usuário, e também praticar como estilizar a interface para que fique boa e fácil de usar. Além disso, é uma boa chance para melhorar nossas habilidades em React e ver como as coisas funcionam na prática.

Habilidades Desonvolvidas: H3, H4, H5, H6, H9 e H10

A atividade tem como objetivo criar e configurar um projeto React.js com Vite, focando no desenvolvimento de um aplicativo que usa a API do GitHub para exibir informações de usuários. Os participantes aprenderão a construir componentes, gerenciar estados, estilizar a aplicação e lidar com dados assíncronos. Além disso, será abordado o tratamento de erros e a renderização condicional, com a execução do projeto localmente para visualizar o resultado.

Habilidades Desenvolvidas: H3, H4, H5, H6, H9 e H10

A atividade busca aprofundar o conhecimento sobre APIs, cobrindo tipos (RESTful, SOAP, GraphQL), ferramentas (Postman, Swagger), verbos HTTP (POST, PUT, GET, DELETE), formatos de dados (JSON, XML), e a diferença entre HTTP e HTTPS. O objetivo é capacitar os participantes a integrar sistemas e usar as ferramentas apropriadas de forma eficaz, além de apresentar e explicar esses conceitos com clareza.
